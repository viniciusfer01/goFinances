### How to run it?

First, have [GoFinances Back-End API](https://github.com/viniciusfer01/goFinances-back-end) running in your machine.

Then,

1. `git clone https://github.com/viniciusfer01/goFinances-back-end.git`
2. `cd goFinances-back-end`
3. `yarn`
4. `yarn start`
